# PreyUI Changelog

This changelog is written for regular users and focuses on what changed in-game.

## 3.0.12 - 2026-04-02

- Fixed a bug that could trigger Blizzard blocked-action errors during combat or while targeting enemies.
- Improved how extra action buttons and zone ability buttons behave so they are more stable during gameplay.
- Continued cleanup work to reduce annoying UI errors and edge-case breakage.

## 3.0.11 - 2026-04-02

- Fixed issues that could cause errors in the Escape menu, quest map, and alert popups.
- Improved general UI stability in Blizzard windows and notifications.
- Synced version information across the addon files and project docs.

## 3.0.10

- Updated the packaged release metadata for the current PreyUI build.

## Notes

- `CHANGELOG.md` is the player-friendly changelog.
- `CHANGELOGS.md` is the more technical/internal change log.
