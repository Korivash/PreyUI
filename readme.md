# PreyUI 

PreyUI is a World of Warcraft Retail UI addon suite focused on clean presentation, strong usability, and a modern layout built for real gameplay. It is designed for players who want their interface to feel sharp, efficient, and powerful without turning their screen into clutter.

Built with both style and function in mind, PreyUI focuses on keeping important information visible, organized, and easy to interact with in all types of content. Whether you're pushing Mythic+, raiding, doing open-world content, or just want a better overall UI experience, PreyUI aims to provide a polished command center that feels good to use every day.

## Why PreyUI Exists

PreyUI exists because a lot of players still want a UI that feels modern, lightweight, and intentional. Many addon setups either become too bloated, too messy, or too inconsistent over time. PreyUI was built to go in the opposite direction.

The goal is to deliver a UI suite that feels clean, aggressive, and visually unified while still being practical for real gameplay. Every part of the interface is meant to serve a purpose. Instead of overwhelming the player with noise, PreyUI is designed to keep the screen readable, responsive, and easy to customize.

PreyUI is currently maintained by **one developer: Korivash**. While this project has its own identity and direction, I still have a lot of respect for Quazii and the impact his UI work had on the community. That influence helped show how strong a clean, high-performance UI could feel in World of Warcraft, and that level of care for design and usability is something I respect deeply.

This project is my way of continuing to build a UI experience that feels polished, useful, and worth maintaining for players who want something clean and reliable going forward.

## Design Philosophy

PreyUI is built around a few core ideas:

- A strong visual identity without sacrificing clarity.
- A clean layout that stays readable during intense combat.
- Customization that feels useful instead of overwhelming.
- Better organization for settings, modules, and utility features.
- A UI that feels modern, responsive, and alive.

The Command Deck is a major part of that experience. Instead of treating settings like an afterthought, PreyUI makes configuration part of the product. The goal is not just to have a lot of options, but to make those options feel good to browse, understand, and use.

## What PreyUI Includes

PreyUI is more than a single window or a small collection of tweaks. It is a full addon suite built to improve the complete gameplay experience.

Included systems and modules currently feature:

- Rebuilt Command Deck configuration shell
- Unit frames and castbars
- Action bar enhancements
- Minimap and datatext modules
- Cooldown manager and class/resource tracking
- Mythic+ tools and utility modules
- Automation and quality-of-life helpers
- Brewmaster stagger bar module

Each module is meant to fit into the same overall visual and functional style, helping the UI feel consistent instead of stitched together.

## Command Deck Highlights

The Command Deck is the core control center of PreyUI. It was built to give the addon a stronger identity and make management of features feel more premium, structured, and accessible.

Highlights include:

- Updated PreyUI branding and metadata
- Header displays the current addon version
- Integrated official Discord link
- Supports repo-based `logo.png` branding
- Drag the window by the top header
- Resize from the bottom-right corner
- Panel size persists between reloads
- Organized layout designed for future expansion

The overall goal of the Command Deck is to make configuration feel clean and intentional instead of cramped or outdated.

## Built for Real Gameplay

PreyUI is made for actual use in live content, not just screenshots. That means the layout and tools are designed to support high-action environments where clarity matters.

Whether you are:

PreyUI is an actively maintained project and will continue evolving over time. The focus is not just on keeping it working from patch to patch, but on continuing to improve how it looks, feels, and functions.

That includes:

- compatibility updates for current Retail patches

The long-term goal is simple: keep PreyUI sharp, reliable, and worth using.

## Slash Commands
- `/prey` opens Command Deck
- `/preyui` alias for `/prey`
- `/rl` safe reload flow
- `/kb` quick keybind mode
- `/cdm` cooldown settings toggle
- `/stagger` stagger module commands

## Installation

### Manual Installation
1. Download the latest release zip.
2. Extract the `PreyUI` folder.
3. Place `PreyUI` in:
   - `World of Warcraft/_retail_/Interface/AddOns/`
4. Launch the game and enable PreyUI.
5. Type `/prey` in chat.
